# 考场信息表 - 使用说明

## 这是个啥？
就是一个显示考试安排的网页，带个大时钟，还能看什么时候考试。

## 有啥用？
- 显示现在几点（自动联网对时 ~~说不定比你的本地时间要准~~）
- 列出所有考试科目和时间
- 自动告诉你考试是“还没开始”、“正在考”还是“考完了”
- 可以自己添加、删除考试信息

## 怎么用？
1. ~~打开该网页~~
2. 网页里可以自己加考试，或者加载写好的数据文件

数据格式（给懂的人看）：
考试信息就是个JSON文件，按固定格式写考试信息就行。
